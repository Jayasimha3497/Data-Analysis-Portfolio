### E-Commerce YOY Growth and Payment Mode Analysis
**File:** [`Ecommerce_Sales_Data_2023_2025.xlxs`](https://github.com/Jayasimha3497/Project-Portfolio/blob/main/Ecommerce_Sales_Data_2023_2025.xlsx)

**Goal:** To analyze and visualize e-commerce sales performance across multiple years,identify trends in customer payments and orders, and evaluate Year-Over-Year(YOY) growth to support data-driven decision making.

**Description:** This project involves a dynamic Excel dashboard using real-world e-commerce transaction data from 2023-2025.The dataset includes order id,order date,customer name,region,city,category,sub-category,product anme,quality,unit price,discount,sales,profit,payment mode(Credit Card,Debit Card,UPI,Cash,COD).

The dashboard provides clear insights into total sales,order volume,average discounts and YOY growth trends by year and payment type(Online/Offline)

**Skils:** 
- Data cleaning,transformation and calculated columns(e.g., Payment Type, Year extraction)
- Used IF,SUMIFS,COUNTIFS, and YEAR formulas for metric and segmentation
- Built multiple Pivot Tables for YOY sales,profit,Order,no. of online/offline payment type,total online sales,total offline sales and region-wise performance
- Created interactive dashboard visuals-KPI cards,line chart,bar chart,column chart,pie chart and slicers
- Compared yearly sales,orders, and profit growth using YOY% formulas and conditional formatting

**Technology:**
- Microsoft Advanced Excel

**Result:**
Built an interactive Excel dashboard summarizing three years of e-commerce performance,delivered actionable insights such as top-performing payment methods and sales growth trends,achieved clear YOY growth visualization with dynamic KPI cards,improving decision-making clarity.
